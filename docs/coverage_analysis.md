# 지식iN 커버리지 갭 분석

> 지식iN 실제 보호자 질문 **80,000건**을 수의사 코퍼스(21,604건)에 검색해 top-1 유사도로 커버리지를 정량화. Gemini 호출 0회 (로컬 BERT 임베딩).

|------|------|----:|----:|
| ✅ 잘 커버 | 유사도 ≥ 0.7 | 74,084 | **92.6%** |
| 🟡 부분 커버 | 0.5 ~ 0.7 | 5,578 | 7.0% |
| ❌ 갭 (미커버) | < 0.5 | 338 | **0.4%** |

## 해석

- 실제 보호자 질문의 **93%**는 우리 수의사 코퍼스로 답변 가능, **0%**는 코퍼스에 유사 사례가 없어 답변 신뢰도가 낮음(데이터 갭).
- 갭 영역을 보강하면 챗봇 커버리지를 직접 끌어올릴 수 있음 → 후속 데이터 수집 우선순위.
