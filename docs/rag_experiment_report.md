## 1. 실험 목적

본 챗봇은 단순 LLM 호출이 아니라 **Sentence-BERT 검색 + Gemini 생성을 결합한 RAG(Retrieval-Augmented Generation)** 구조다.
"왜 LLM 단독이 아니라 검색을 결합했는가"를 정량·정성적으로 검증하기 위해, 동일한 보호자 질문에 대해 세 가지 답변 방식을 비교했다.

| 방식 | 설명 | 검색 | 생성 |
|------|------|:--:|:--:|
| **A. Gemini 단독** | 검색 없이 LLM만으로 답변 (baseline) | ✗ | ✓ |
| **B. BERT 검색 원본** | 수의사 코퍼스에서 검색한 top-1 답변 원문 | ✓ | ✗ |
| **C. RAG (현재 챗봇)** | BERT 검색 결과를 근거로 Gemini가 생성 | ✓ | ✓ |

## 2. 실험 설계

- **테스트 입력**: Naver 지식iN에서 수집한 **실제 보호자 질문 12건** (intent별 4건: 증상/처치/응급), 길이 40~160자, seed 고정 재현 가능
- **참고 답변(reference)**: 세 방식 모두 BERT가 수의사 Q&A 코퍼스(21,604건)에서 검색한 답변을 기준으로 사용
- **평가 지표 4종**

| 지표 | 정의 | 의미 |
|------|------|------|
| 질문 관련성 | cos(질문, 답변) 임베딩 유사도 | 답이 질문에 부합하는가 |
| 수의학 근거성 | max cos(답변, 검색 참고풀 top-20) | 답이 실제 수의학 지식에 기반하는가 (B는 자기자신 top-1 제외) |
| 환각율 | 참고답변에 근거 없는 단정적 의학 주장 비율 (Gemini LLM-as-judge) | 답이 근거 없이 지어낸 정도 |
| 답변 길이 | 글자 수 | 구체성·완결성 proxy |

## 3. 결과

### 3.1 정량 결과 (전체 평균, n=12)

| 방식 | 질문 관련성 | 수의학 근거성 | 환각율 | 답변 길이 |
|------|:----------:|:------------:|:----:|:--------:|
| A. Gemini 단독 | **0.801** | 0.787 | 25.0% | 243자 |
| B. BERT 검색 원본 | 0.669 | **0.818** | 0.0% | 405자 |
| C. RAG | 0.718 | 0.802 | 27.5% | 184자 |

### 3.2 정성 분석 — 환각 vs 근거 기반 교정

평균 환각율과 별개로, **A·C 환각 격차가 큰 케이스에서는 RAG가 명확히 우위**를 보였다.

**예시 1) 췌장염 → 하반신 마비 인과 질문**
- A. Gemini 단독: "하반신 마비까지 이어지는 경우는 흔치 않습니다" → 모호, 근거 불명
- C. RAG: **"췌장염 자체로 직접적인 하반신 마비를 일으키는 경우는 드물답니다"** → 검색된 수의사 답변 근거로 잘못된 인과를 명확히 교정

**예시 2) 강아지 기침 영양제 질문**
- A. Gemini 단독: 일반론으로 길게 서술 (judge 환각 100%)
- C. RAG: 검색 근거 내에서 "특정 기침 영양제보다 호흡기·면역 제품" 으로 한정 (judge 환각 0%)

## 4. 해석

- **수의학 근거성**: 자기자신 제외 보정 후에도 **RAG(0.802) > Gemini 단독(0.787)**. 검색이 답변을 실제 수의학 지식에 더 근접시킴을 확인.
- **질문 관련성**: Gemini 단독(0.801)이 가장 높으나, 이는 **질문 문장의 어휘를 그대로 반복**해 표면 임베딩 유사도가 올라간 결과다. 정성 분석에서 보듯 내용 정확성과는 별개이며, RAG는 짧고 정제된 답변(184자)으로 핵심만 전달한다.
- **환각율**: 정성 사례에서는 RAG의 교정 효과가 뚜렷하나, **전체 평균에서는 RAG가 오히려 높게 측정**되었다 (아래 한계 참조).

## 5. 한계 (반드시 명시)

1. **표본 크기**: n=12로 작아 지표, 특히 환각율의 분산이 크다. RAG 답변은 짧아(184자) 의학 주장이 1~2개뿐이라, 주장 하나만 걸려도 비율이 50~100%로 급변한다 → 평균 환각율(A 25% vs C 27.5%)의 역전은 통계적 신뢰도가 낮다.
2. **LLM-as-judge 노이즈**: 환각 채점을 Gemini로 자동화해 채점 일관성이 부족하다. 동일 품질 답변에도 케이스별로 0%·100%가 혼재.
3. **관련성 지표 편향**: 코사인 유사도는 어휘 중복에 민감해, 질문을 반복하는 장황한 답변(Gemini 단독)에 유리하게 작용한다.

## 6. 결론 및 후속 과제

- RAG의 가치는 **① 수의학 근거성 향상(정량 확인)** 과 **② 잘못된 인과·단정의 근거 기반 교정(정성 확인)** 에서 드러난다.
- 다만 **현 표본(n=12)으로 환각 감소를 정량 입증하기엔 부족**하다.
- **후속 과제**: 테스트 질문을 50~100건으로 확대하고, 환각 채점에 다수결(여러 judge) 또는 사람 라벨을 도입해 환각율의 신뢰구간을 확보한다.
