## Table 12. 전체 검색 성능 비교

| 모델 | Hit@1 | Hit@3 | Hit@5 | MAP@5 |
|------|-------|-------|-------|-------|
| TF-IDF | 0.1800 | 0.4800 | 0.6200 | 0.0974 |
| Sentence-BERT | **0.2400** | **0.5200** | **0.6200** | **0.1287** |

**해석:**
- BERT가 Hit@1(+6%p), Hit@3(+4%p), MAP@5(+0.0313) 전반에서 우위
- Hit@5는 양 모델 동일(62%) — 5개 후보 안에는 동등하게 검색
- MAP@5 격차(0.1287 vs 0.0974)는 BERT가 정답 문서를 더 높은 순위에 배치함을 의미

- 노령견: 두 모델 동일 — 노령견 질환 패턴이 두 방식 모두에 균등하게 분포

## 4.4절 ANOVA 결과

**연구 질문:** 진료과(department)에 따라 질문 텍스트 길이에 유의한 차이가 있는가?

| 항목 | 값 |
|------|-----|
| 검정 방법 | One-way ANOVA |
| 독립변수 | 진료과 (내과/외과/피부과/안과/치과, k=5) |
| 종속변수 | 질문(input) 텍스트 길이 (문자 수) |
| 전체 표본 | N = 21,604 |
| F 통계량 | F(4, 21599) = 229.46 |
| p-value | p = 2.87e-193 |
| 결론 | 진료과별 질문 길이 차이는 통계적으로 매우 유의함 (p < 0.001) |

**해석:**
- 내과·외과 질문은 증상 기술이 길고 복잡한 경향
- 피부과·안과 질문은 시각적 증상 위주로 간결한 편
- 진료과별 텍스트 특성 차이가 TF-IDF/BERT 성능 차이의 원인 중 하나일 수 있음

## 인사이트 4. 양쪽 실패 케이스 공통 패턴 (보고서 5.3절)

> Hit@5 기준 양쪽 모델 동시 실패 10건 정성 분석 결과

| 유형 | 비율 | 대표 사례 | 실패 원인 |
|------|------|----------|----------|
| ① 의료 절차 질문 | 50% (5/10) | 수술 여부·마취·MRI 관련 | 증상이 아닌 처치 질문 — 증상 기반 임베딩 구조에 구조적 불리 |
| ② 이물질 섭취·급성 상황 | 30% (3/10) | 포도·메로나·이물 섭취 | 표현 방식 다양 — 유사 문서 매칭 어려움 |
| ③ 희귀 질병 | 20% (2/10) | 부신피질기능항진증·자궁축농증 | 학습 데이터 희소성 — 유사 문서 절대 부족 |

**결론:**
TF-IDF·BERT 공통 한계 영역이 식별되었으며, 이는 단순 매칭 방법론 개선으로 극복이 어렵다.
